ecological what?

The skill we're learning is problem solving not cartography. Cartography is an art and science in itself. We're using a set of tools to make maps, and I encourage you to learn more about GIS, cartography and data visualization.

These tools are going to change, and I want you to be ok with that.

You are going to encounter problems when you make maps that you and I have never seen before. I want you to be able to work through those problems on your own, without feeling lost. Part of that is learning how to get help. Learn how to ask better questions, and who to ask. Learn where to look for documentation. Learn how to prompt a large language model. Learn when to ask your editor for more time.

Some other tools you might've used, in other contexts: D3, QGIS, Illustrator, pen and paper.
